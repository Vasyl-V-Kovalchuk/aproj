Перелік файлів:

-	proj – папка що містить проект;

-	screens
-	routes
-	models
-	public – програма, що виконується;
-	src – код серверної частини, в цьому каталозі зберігається вихідні проекту, містить файли з вихідним кодом на мові js;
-	components - папка що містить файли всіх компонентів додатку.S
-	hooks - папка що містить файли всіх веб-хуків
-	demo - папка с відеоінструкцією
